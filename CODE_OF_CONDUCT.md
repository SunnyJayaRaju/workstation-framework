# Code of Conduct

## Our Commitment

We are committed to providing a respectful, welcoming, and collaborative environment for everyone.

## Expected Behavior

- Be respectful.
- Be constructive.
- Welcome feedback.
- Help others learn.
- Focus on technical discussions.

## Unacceptable Behavior

Examples include:

- Harassment
- Personal attacks
- Discrimination
- Trolling
- Spam
- Publishing private information without permission

## Enforcement

Project maintainers may edit, remove, or reject contributions that violate this Code of Conduct.

Serious or repeated violations may result in temporary or permanent restrictions from participating in the project.

## Contact

If you believe someone has violated this Code of Conduct, please open a private GitHub Security Advisory or contact the project maintainer.
